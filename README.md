## Setup

- buka direktori project di terminal anda.
- ketikan command : cp .env.example .env (copy paste file .env.example)
- buat database
- Lalu ketik command dibawah ini

- composer install
- php artisan optimize:clear
- php artisan key:generate (generate app key)
- php artisan serve
